Kawane's Songbook

Songbook is an extremely simple server that allows you to store and edit songs with chords and lyrics on your own server. It helps setting you free of web site full of distracting animations to focus on the song.

It aims to be easily installed on a server, including for not technical users.

Developed withJava IDE with advanced HTML/CSS/JS

Features

SongBook is fast and easy to use. Songs are directly editable. User management couldn't be simpler.

Store songs with lyrics, chords and some meta-data all in pure html. Full text search over all songs including lyrics.

There is no user management, no administration, only a minimal access control.

Songs are editable in Wysigiw. It's fully compatible with smartphone and tablets.

Get your server

SongBook can easily installed on a server (with Java 8) or a Paas (OpenShift, ...).
